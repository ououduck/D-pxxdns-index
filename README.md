## ✨ 特点

- 🔄 **动态数据加载** - 不同于其他固定模板，实时从数据库中获取站点信息
- 🎨 **企业级设计** - 专业美观的界面设计
- ⚡ **即插即用** - 简单快速的部署流程

---

## 📦 安装使用

### 步骤 1：下载项目

下载本项目的 `.zip` 整合文件

### 步骤 2：上传解压

将 `.zip` 文件上传到 `网站目录/public/template` 目录下并解压

### 步骤 3：验证目录

确保目录下存在 `D-pxxdns-index` 文件夹

### 步骤 4：启用模板

进入 PXXDNS 系统后台：**后台 → 系统 → 模板配置 → 点击启用**
