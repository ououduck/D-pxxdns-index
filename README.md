# D-pxxdns-index
## 为PXXDNS定制的企业级首页模板  
![D-pxxdns-index](https://fogpic-vip.3pw.pw/20260222/61b1546526fe56be9cfd49ee286268f9.png)
### 特点：不同于其他固定模板 实时从数据库中获取站点信息
  
## 使用方法  
### 下载本项目的.zip整合文件  
### 将.zip文件上传到 网站目录/public/template 目录下并解压  
### 确保目录下有一个 D-pxxdns-index 的文件夹
### 最后到PXXDNS系统的 后台》系统》模板配置》点击启用即可
